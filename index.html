Pipiduwa

Pipiduwa is a social network dedicated to video games. It lets users follow friends, create and manage lists (for example, to-do or favorites), and search for games to find details such as platform and rating.

The name “Pipiduwa” comes from Polish "pipidówa" (pronounced “pipidoova”) and refers to a small, isolated place — fitting for a project that began as a personal solution: a platform to record the games I’ve played, keep track of titles I want to play, and share them with friends.

The frontend is built with Vue.js; the API is a RESTful Spring Boot service that supplies data and authenticates users via Keycloak; data is stored in MongoDB and the media are stored in a S3 bucket. The project also includes a scraping service that populates Pipiduwa’s database with information from IGDB and RAWG.
