Pipiduwa

Pipiduwa is a social network dedicated to video games. It lets users follow friends, create and manage lists (for example, to-do or favorites), and search for games to find details such as platform and rating.

The name "Pipiduwa" comes from Polish "pipidówa" (pronounced "pipidoova") and refers to a small, isolated place — fitting for a project that began as a personal solution: a platform to record the games I've played, keep track of titles I want to play, and share them with friends.

The frontend is built with Vue.js; the API is a RESTful Spring Boot service that supplies data and authenticates users via Keycloak; data is stored in MongoDB and the media are stored in a S3 bucket. The project also includes a scraping service that populates Pipiduwa's database with information from IGDB and RAWG.

Chat Ollama

See the Github repository

Chat Ollama is a web-based interface designed for Linux users like me who don’t have access to a native graphical app but still want to interact with Ollama. It provides a lightweight, intuitive, and accessible way to communicate with local models directly through the browser or in a desktop application, making AI interaction seamless even in Linux-only environments.

To install it as a desktop application, you may use the install.sh script. This script uses the Electron-Builder tool to first create the AppImage, then copies the icon, symbolic link, and AppImage to the appropriate folders. Installing and uninstalling is pretty intuitive.

Chat with any Ollama service

This app provides a simple web interface using a standalone Vue.js script. It can be run in a browser via a local web server or as a desktop app, depending on your needs.

By default it will try to connect to your local machine's instance but you can edit the address as needed.It will automatically detect all the models currently installed on your device and you can pick it easily.
